AI AFTER — ПОЛНАЯ ИНСТРУКЦИЯ ПО ДЕПЛОЮ
==========================================

ЧТО ВПИСЫВАТЬ В КЛЮЧИ (Vercel -> Settings -> Environments ->
Production -> Add Environment Variable). Для каждого: слева поле
Key — впишите ровно то, что написано ЗАГЛАВНЫМИ БУКВАМИ ниже,
справа поле Value — само значение с соответствующего сайта.

1) GEMINI_API_KEY
   Взять на: aistudio.google.com/apikey (без карты)
   Значение: ключ, который дал сайт (длинная строка)

2) UPSTASH_REDIS_REST_URL
   Взять на: upstash.com -> создать Redis-базу -> вкладка REST
   Значение: похоже на https://xxxxx.upstash.io

3) UPSTASH_REDIS_REST_TOKEN
   Там же, рядом с URL (кнопка "глаз", чтобы увидеть)
   Значение: длинная строка-токен

4) GROQ_API_KEY  (необязательно, резервный ИИ)
   Взять на: console.groq.com/keys (без карты)

5) GOOGLE_CLIENT_ID  (необязательно, вход через Google)
   Взять на: console.cloud.google.com
   -> создать проект -> APIs & Services -> Credentials
   -> Create Credentials -> OAuth client ID -> Web application
   -> в "Authorized JavaScript origins" впишите ваш домен,
      например https://afteraibot.vercel.app
   -> скопируйте Client ID (заканчивается на .apps.googleusercontent.com)
   Если не настраивать — просто не задавайте эту переменную,
   кнопка входа через Google не появится, всё остальное работает.

После добавления всех переменных — Save, затем на вкладке
Deployments у последнего деплоя ⋯ -> Redeploy.

КАК ЗАДЕПЛОИТЬ С НУЛЯ
-------------------------
1. vercel.com/drop -> перетащить архив -> дать имя -> Deploy
2. Прописать ключи выше
3. Redeploy

ЧТО НОВОГО В ЭТОЙ ВЕРСИИ
----------------------------
- Все смайлики убраны, заменены на чистые иконки
- Профиль теперь в сайдбаре: аватар листается стрелками влево/вправо
  (3 готовых персонажа), клик по аватару открывает настройки
- Пароли хешируются на сервере (bcrypt) — больше не хранятся открытым
  текстом, вся логика входа/регистрации ушла в api/auth.js
- Свой красивый календарь (год -> месяц -> день) вместо стандартного
- Вход через Google (см. пункт 5 выше, полностью опционально)
- Логотип AFTER AI в шапке и на экране входа
- Синяя тема по умолчанию
- Кнопка Telegram встроена в шапку (больше не перекрывает поле ввода)
- Характер бота: тёплый, с мнением, но профессиональный —
  без занудства и без подхалимства
- Файлы: бот создаёт настоящие скачиваемые PDF/TXT/любые файлы
  отдельной карточкой с просмотром
- Новостная лента с колокольчиком и счётчиком непрочитанных
- День рождения — конфетти при входе в нужный день

ВАЖНО
-------
- НЕ прикладывайте package.json к ручной загрузке через vercel.com/drop
  без интернета — при первом деплое Vercel сам поставит зависимость
  bcryptjs, это может занять на 10-20 секунд больше обычного, это
  нормально.
- Все токены/ключи — секретные, никому их не показывайте и не
  вставляйте в чат.
